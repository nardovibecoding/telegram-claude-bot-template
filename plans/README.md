# Plans

Active and completed plans. Each plan is a markdown file with checkboxes.

## Format
```markdown
# Plan: [title]
Status: active | paused | completed
Created: YYYY-MM-DD
Updated: YYYY-MM-DD

## Goal
One sentence.

## Steps
- [x] Done step
- [ ] Pending step
- [ ] Future step

## Notes
Context, decisions, blockers.
```
